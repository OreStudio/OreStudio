* Goal

Get the Windows package installable on a real Windows machine, fix
the local-time / shutdown-crash regressions surfaced along the way,
and clear up assets packaging.

* Acceptance

- Windows package installs and runs (console end-to-end).
- Windows clang build compiles =-Werror= clean (no =localtime=
  deprecation).
- Qt UI shuts down without crash.

* Decisions

- /Adopt =X_VCPKG_APPLOCAL_DEPS_INSTALL=true=/ :: copies Qt and
  other DLLs into the bin folder during install; closes the bulk of
  the Windows runtime-dependency failures.
- /=safe_localtime= helper/ :: Windows uses =localtime_s=, the rest
  use =localtime_r=. Behind one function so call sites stay simple.
- /Sole-ownership rule for detachable MDI widgets/ :: fixes the
  shutdown crash; widgets stay parent-less, =MainWindow= owns them,
  =QObject::destroyed= cleans the lookup maps.

* Out of scope

- macOS code signing.
- Windows installer beyond the basic vcpkg-driven packaging.
