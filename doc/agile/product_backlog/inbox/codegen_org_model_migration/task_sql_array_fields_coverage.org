#+title: Task: Extend converter + org_loader for array-structured fields

This task adds sub-heading / table-based org representations for each kind, extends both ends of the pipeline, and re-emits the affected org files so --profile sql round-trips cleanly and the Qt UI generator emits static combo box values.
