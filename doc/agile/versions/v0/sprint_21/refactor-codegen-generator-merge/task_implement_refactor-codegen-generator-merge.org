* Goal

Move all logic from the top-level =src/generator.py= monolith into
=src/codegen/core.py=. Remove the =_import_generator= dynamic-import hack in
=codegen/generate.py= (which manipulates =sys.path= and does =import generator=
at runtime). After the move, =generate.py= imports from =.core= like every other
module in the package. Delete =generator.py= once empty.

This eliminates a structural coupling between the package and the top-level
script, makes the codebase importable without side-effects, and allows standard
tooling (mypy, pytest, editors) to resolve all symbols without path hackery.
