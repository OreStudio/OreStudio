* Goal

Make a sweep through the Qt UI to fix the small issues that would
compound as we add more widgets: launcher integration, logging
wiring, column header names, row sizing, tab opening behaviour, and
the main-window-following table size.

* Acceptance

- Prodigy launcher integrated.
- Logging is wired into the UI.
- Column headers display names.
- Row heights are sensible.
- Selecting "Currencies" from the menu opens the Currencies tab.
- The currencies table follows the main window size.
