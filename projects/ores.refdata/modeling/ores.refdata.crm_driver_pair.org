A single [[id:1907531F-E2AF-4BF7-84A4-6D69CB9EDFD7][spanning-tree]] edge for its parent =crm_topology_config=: the
directly-quoted pair a producer publishes ticks for. The enabled rows for
a (tenant, party) are read by =ores.marketdata.service= and fed straight
into =ores.analytics.quant::topology_builder::build= as its
=ccy_pair_input= list -- see [[file:../../../doc/agile/versions/v0/sprint_23/crm_implementation/task_wire_crm_into_marketdata_ingest.org][the wiring task]].

Scoped to a tenant and a party, matching its parent config.

*** read_latest_all_tenants
:PROPERTIES:

Reads the latest CRM driver pairs across all tenants. Used by
=crm_ingest_bridge= at startup/refresh, which must build a
=rate_engine= for every (tenant, party) regardless of tenant -- there is
no single tenant context to scope the read to.

#+begin_src cpp :name declaration :implements DCA78C69-E508-48D9-9972-A9B8094D91FB
std::vector<domain::crm_driver_pair> read_latest_all_tenants(context ctx);
#+end_src

#+begin_src cpp :name implementation :implements F2EB1914-5E94-42CA-9C77-46BDB364BF9E
std::vector<domain::crm_driver_pair>
crm_driver_pair_repository::read_latest_all_tenants(context ctx) {
    static const auto max(make_timestamp(MAX_TIMESTAMP, lg()));
    const auto query = sqlgen::read<std::vector<crm_driver_pair_entity>> |
                       where("valid_to"_c == max.value()) |
                       order_by("id"_c);

    return execute_read_query<crm_driver_pair_entity, domain::crm_driver_pair>(
        ctx,
        query,
        [](const auto& entities) { return crm_driver_pair_mapper::map(entities); },
        lg(),
        "Reading latest CRM driver pairs across all tenants");
}
