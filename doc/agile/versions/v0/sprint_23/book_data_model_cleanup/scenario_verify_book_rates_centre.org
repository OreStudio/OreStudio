** Books list shows the Rates Centre column with a flag icon

Open Reference Data > Books. Confirm a "Rates Centre" column appears
after Sweepable, rendering blank (no flag) for every seeded book —
the field is nullable and no seed data sets it — with no error
banner.

** Detail dialog shows and edits the Rates Centre combo

Double-click any book. Confirm a "Rates Centre:" combo box appears
(blank/unselected initially). Open the dropdown, confirm it is
populated with business centre codes each showing a small flag icon
(e.g. =GBLO=, =USNY=). Select one, Save. Confirm the change-reason
prompt appears, the save succeeds, the grid refreshes, the list's
Rates Centre column now shows the matching flag icon, and Version
increments by 1.
