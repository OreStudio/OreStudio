** Export CSV with the full unfiltered grid

Click "Export CSV", save to a temp path. Open the file and confirm:
one header row (=base,quote,rate,status,as_of=), one data row per
non-dash cell actually shown in the grid (diagonal and "no data"
dash cells excluded), values match what's on screen.

** Export ORE with the full unfiltered grid

Click "Export ORE", save to a temp path. Open the file and confirm:
tab-delimited =DATE KEY VALUE= lines, one per fresh/stale cell shown
in the grid (unavailable-status cells excluded), key format
=FX/RATE/<BASE>/<QUOTE>=, date matches the cell's as_of date, value
matches the displayed rate.
