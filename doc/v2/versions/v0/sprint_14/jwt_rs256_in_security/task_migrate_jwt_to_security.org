* Goal

Move JWT infrastructure from =ores.http= to =ores.security= and switch to RS256 asymmetric signing so services can validate tokens with just a public key.

* Acceptance

- JWT moved from =ores.http= to =ores.security= (shared across services).
- RS256 asymmetric signing: IAM mints tokens with private key; other services verify with public key.
- =jwt_claims= extended with =tenant_id= + =party_id= for distributed service auth.
- =ores.iam.service= mints + returns RS256 JWT in =login_response=.
- Protocol 49.0 (breaking).

* Plan

Captured during execution; cleared into the parent story on close.

* Notes

Prerequisite for the NNG broker architecture; survived the NATS migration because the security model is independent of the transport.

* Result

JWT centralised in ores.security; RS256 in use.
