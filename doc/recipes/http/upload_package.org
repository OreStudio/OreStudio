Upload an engine package bundle (tar.gz) for a given app version. The
=app_version_id= is the UUID of the app version record.

#+begin_src verb :wrap src text
post http://localhost:8080/api/v1/compute/packages/{{(verb-var app_version_id)}}
Content-Type: application/octet-stream

<<engine-package.tar.gz
