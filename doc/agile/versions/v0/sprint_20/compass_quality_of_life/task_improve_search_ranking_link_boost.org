* Goal

doc_index already tracks inbound links per node. Surface more-authoritative docs higher in compass search by applying a link-count boost to FTS rank. Stories linked by many tasks should float above loosely-connected docs with the same keyword density. Design the weighting function carefully so hub docs (glossary, sprint manifest) do not over-dominate; type-weighted links (task→story vs. recipe cross-reference) may help. Validate that the boost does not hurt question-shaped queries where FTS relevance is already good.
