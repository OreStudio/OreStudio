* Result

Abandoned. The =#+has_qt: false= flag was a single-purpose boolean that would have proliferated
as new facets were added. Replaced by [[id:86510CDC-7543-4B18-A6DC-8EA3E1AF8A8F][B5]], which implements the generic MASD address-based
mechanism: =:ores.cpp.qt.enabled: false= in the entity's =:PROPERTIES:= drawer disables the
Qt facet without any new code paths or flags. See [[id:43FF3591-AD18-4CE3-8239-A220862C9B5A][Applied MASD]] and [[id:0BD6F9EC-34DD-44D9-8AA1-917EE5966396][ORE Studio Variability Model]].
