* Goal

The =Product website= workflow builds the Org site exactly once per push
to main. The =build= job uploads the rendered site as an artifact and the
=deploy= job downloads it, layers Doxygen on top, and deploys — instead of
rebuilding the site from scratch. This removes the ~5 min duplicate build
from every main merge while keeping the Org build outside the serialized
=pages= concurrency group.
