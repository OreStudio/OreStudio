** Connect to tenant Barclays Plc

Log in as =tenant_admin@barclays_plc= / =Secure-Password-123= and
select *BARCLAYS PLC*.

** Open Interest Rate Curves

- Menu: Market Data > Interest Rate Curves.
- Confirm a window titled "Interest Rate Curves" opens listing rows
  for multiple currencies (e.g. RATES / YIELD / USD/SOFR, RATES /
  YIELD / EUR/ESTR) -- this list is the official market_series
  catalog, not anything synthetic-specific; it would show the same
  way for a real vendor feed.

** Open the Snapshot window for USD/SOFR

- Double-click the *USD/SOFR* row.
- Confirms pass: a new MDI sub-window titled "Curve Snapshot: USD/
  SOFR" opens (not a modal dialog -- the Interest Rate Curves window
  should still be usable behind it), with *Grid* and *History* tabs.

** Verify the Grid tab

- On the *Grid* tab: confirm a table on the left with columns Tenor /
  Rate / Observed, one row per tenor (e.g. 1M, 3M, 6M, 12M, 2Y, 5Y,
  10Y), in ascending tenor order.
- Confirm a line chart on the right showing the same points, tenor
  labels on the X-axis, roughly matching the table's shape.
- If the feed has not published yet, confirm the empty-state message
  is shown instead of a blank table/chart.
- Click the toolbar *Reload* button; confirm the table/chart refresh
  without closing the window.

** Verify the History tab

- Switch to the *History* tab.
- Set *Bucket size* to a small value (e.g. 1 minute) and *Show last*
  to 5, so buckets are quick to observe.
- Confirm a delta table on the left: one column per bucket, each
  tenor's rate plus a =(+/-X.X)= basis-point delta versus the
  previous bucket (the oldest shown bucket has no delta).
- Confirm an overlay chart on the right: one line per bucket, the
  newest bold/opaque and older ones visibly fading.
- Change the bucket size/count controls; confirm the table and chart
  both refresh automatically (no separate reload needed for History).

** Verify Snapshot for a second currency

- Return to the Interest Rate Curves window (still open behind the
  snapshot); double-click a different row (e.g. EUR/ESTR).
- Confirm a second, independent Curve Snapshot window opens for
  EUR/ESTR, correctly labelled, not reusing/overwriting the USD/SOFR
  window's data.
