* Goal

End duplication of doc-handling between [[id:08FBD248-257A-A474-DD43-DB08949978F1][ores.codegen]] and =ores.compass=
by drawing a clean line: /codegen generates, compass navigates/. Move
the doc-navigation code (=doc_index=, =doc_list=, =doc_show=) out of
codegen into compass as the =list= and =show= commands; leave document
/generation/ in codegen (compass will call it as a library for the
future Scaffold pillar). Add recipes covering compass usage.
