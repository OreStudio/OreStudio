* Goal

From compass, get a quick overview of all currently executing (and recently completed) GitHub Actions runs across the repo's workflows, at a glance see which ones are failing, and for failing runs pull the underlying error detail via the gh CLI -- without having to download the full job log.

* Acceptance

- A compass command shows a summary table of GitHub Actions runs (workflow, branch, status, conclusion) across the repo, similar in spirit to compass pr / compass fleet.
- Failing runs are visually distinguished from passing/in-progress ones.
- For failing runs, the command surfaces the actual failure reason (e.g. via gh run view --log-failed or gh api annotations) without shelling out to download and grep the full raw log file.
- Documented with a recipe under doc/recipes/compass/.
