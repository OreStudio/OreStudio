* Result

Delivered on =feature/trim-bearings-limit-heading=.

=compass bearings= changes:

- =-n/--limit N= (default 5) caps how many items are shown in the recipes
  and memories sections; when truncated a "… N more (compass list …)" line
  is printed.  Pass =-n 0= to restore the old unlimited behaviour.
- =--no-heading= suppresses the new heading section for callers that want
  raw orientation without a recommendation.
- New final section "🎯 Suggested next action" calls =compass heading
  --count 1 --no-banner= so the session's top recommended next step appears
  at the end of every =bearings= run without duplicating the heading banner.

=compass heading= changes:

- =--no-banner= (suppressed from =--help=) omits the ="🧭 ores.compass —
  heading"= header line when called from a parent command.
