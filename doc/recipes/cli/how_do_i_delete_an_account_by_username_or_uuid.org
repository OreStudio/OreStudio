Delete an account by username or UUID. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I delete an account by username or UUID using =ores.cli=?

* Answer

** Delete by username

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli iam accounts delete ${db_args} ${log_args} \
  --key newuser4
#+end_src

#+RESULTS:
#+begin_src json
Account deleted successfully: 3b6efad4-039f-4dd4-8e0d-6209a08f8148
#+end_src

** Delete by UUID

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli iam accounts delete ${db_args} ${log_args} \
  --key 9dec6b76-f982-47c3-bb11-58b62ca5c0d1
#+end_src

#+RESULTS:
#+begin_src json
Account deleted successfully: 9dec6b76-f982-47c3-bb11-58b62ca5c0d1
#+end_src
