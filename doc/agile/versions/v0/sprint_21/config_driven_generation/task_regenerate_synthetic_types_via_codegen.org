* Goal

Author a codegen entity+table model for each of market_data_generation_config, fx_spot_generation_config and gmm_component, and regenerate their backend layers (domain, SQL, JSON/table IO, mapper, repository, service list/save/delete handlers, protocol, generator) via run_generator.sh --profile all, replacing the hand-written equivalents. The bespoke synthetic logic (simulate endpoint, feed_controller, fx_spot_feed) and the Market Simulator UI are preserved and reconciled against the regenerated layers. Qt entity layer stays hand-written for a later task.
