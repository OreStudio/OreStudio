Add a new workunit to an existing compute batch via =ores.cli=. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I add a new workunit to a compute batch?

* Answer

#+begin_src sh :wrap src text
export ORES_CLI_DB_PASSWORD
./ores.cli compute workunits add ${db_args} ${log_args} \
  --batch-id "00000000-0000-0000-0000-000000000000" \
  --app-version-id "00000000-0000-0000-0000-000000000000" \
  --input-uri "/api/v1/compute/workunits/00000000-0000-0000-0000-000000000000/input" \
  --config-uri "/api/v1/compute/workunits/00000000-0000-0000-0000-000000000000/config" \
  --priority 10 \
  --target-redundancy 1 \
  --modified-by admin
#+end_src
