This page catalogues every [[id:B555D2A6-10CA-4A93-973E-F2AF4E1D7E55][runbook]] — named, repeatable compositions
of [[id:0C57932D-D13A-480F-B426-49525AA8BA3D][recipes]] and [[id:6054CC20-5F41-482F-A587-759512577B1D][skills]] into complete multi-step
procedures. For the conceptual overview of how runbooks fit with
instructions, recipes, skills, functions, settings, and memory, see
[[id:94F9949A-1580-4D47-9331-AA0597543C99][LLM Configuration and Artefacts]].

To create a new runbook: =compass add runbook= or
=projects/ores.codegen/generate_v2_doc.sh --type runbook=. List
all runbooks with =compass list --type runbook=.
