* Question

How do I add a new login info entry for an account?

* Answer

#+begin_src sh :wrap src text
export ORES_CLI_DB_PASSWORD
./ores.cli iam login-info add ${db_args} ${log_args} --account-id "e7bddd23-c004-4652-99e1-8d4ff5856fe1"
#+end_src
