* Goal

is actually the Basel III/IV FRTB trading book / banking book
classification -- a crucial regulatory concept, not a bare flag. Scope
grew from a type fix (int -> bool) into modeling it properly: a
regulatory_book_type lookup entity (Trading/Banking) with full Qt CRUD,
badge-coloured combo, and a dedicated knowledge note on the Basel
boundary.
