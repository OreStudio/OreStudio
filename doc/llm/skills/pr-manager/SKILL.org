* When to use this skill

When you have local commits ready and need to drive them through a
PR — title, body, push, monitor, merge.

* How to use this skill

1. /Generate the title and body/ from your commits — [[id:1882D6DF-4012-4C6E-BF62-0D943EF805B0][How do I
   generate a PR summary?]].
2. /Open the PR/ via =gh pr create= — [[id:CC928252-E0C5-4C3C-A7A8-D613593E4E37][How do I create a PR?]].
3. /Hand off to the watch loop/. The post-creation phase
   (monitoring CI, addressing review comments, fixing failures) is
   the [[id:57B7D4D9-AAA5-427D-8D16-0E82DA9D3763][PR Shepherd]] skill. Invoke it; do not embed the loop here.
4. /Merge once green/ and after explicit user confirmation — [[id:95D560C6-3551-4810-9D11-55F9F82B483C][How
   do I merge a PR?]]. The =--delete-branch= flag handles cleanup.
5. /If more phases follow/ on the same feature, transition cleanly
   to the next branch via [[id:6DAFFC2F-7F94-403B-87D0-2A56C4669B7F][Feature Branch Manager]].
