*** Create

Click *Add*. Enter code =TestContact=, name =Test Contact Type=,
description =Scenario test row=, display order =99=. Save. Confirm
the new row appears in the list with the values just entered.

*** Read

Double-click the =TestContact= row to reopen its detail dialog. Confirm
every field round-trips exactly as saved (code, name, description,
display order).

*** Update

Edit the =TestContact= row's description to =Scenario test row
(updated)= and save. Confirm the list refreshes with the new
description without a manual reload, and that a change-reason prompt
was shown and recorded.

*** Delete

Select the =TestContact= row and click *Delete*, confirming the
prompt. Confirm the row disappears from the list.

*** History

Reopen the =TestContact= row via History (not the main list, since its
current version is deleted) and open the History dialog. Confirm all
three prior operations — create, update, delete — are listed in order
with the change reasons entered above.
