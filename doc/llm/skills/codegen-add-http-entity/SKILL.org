* When to use this skill

When you need to add HTTP REST API endpoints for a new entity in
=ores.http.server=. Prerequisites: the domain type must exist ([[id:B1450696-8C51-4CC5-8910-84912A411AB6][domain-type-creator]]),
the service layer must be implemented, and JSON serialisation via =rfl=
must be available. See [[id:608675E1-4284-4902-99AC-4DA565390AFB][Entity lifecycle]] for the full layer ordering.

* How to use this skill

The work is split into three phases, each ending in a PR.

1. /Gather requirements/: entity name, domain, operations needed
   (list/save/delete/history), authorisation (=.auth_required()=,
   =.roles(...)=), OpenAPI tags and summaries.

2. /Phase 1 — list + save/: create or extend the routes class header
   and implementation; register routes in =application.cpp=. PR title:
   =[http] Add {entity} list and save endpoints=.

3. /Phase 2 — delete + history/: add delete (batch) and history
   handlers; register routes. PR title:
   =[http] Add {entity} delete and history=.

4. /Phase 3 — recipe docs/: add =how_do_i_*.org= recipes under
   [[proj:doc/recipes/http][=doc/recipes/http/=]]. PR title: =[doc] Add {entity} HTTP endpoint recipes=.

Use =feature-branch-manager= between phases and =pr-manager= to open each PR.
Build after every step (via =compass build=, not raw =cmake --build=, so it takes the host-wide build lock — see [[id:F176FCA1-68D5-420D-B076-75A65FEE5EBB][How do I build the system?]]): =./compass.sh build ores.http.server.lib=.

* Codegen gap

❌ No codegen profile exists for =ores.http.server=.  Do not hand-code
this layer.  When [[id:0CBA2F23-7F5F-433D-921A-D9C056BD9CA8][entity-creator]] reaches this layer:

1. Raise a sprint story: "Implement codegen profile for HTTP endpoints."
2. Reference [[id:608675E1-4284-4902-99AC-4DA565390AFB][Entity lifecycle]]§"Codegen gaps" for what the profile must cover.
3. Link the story to the entity task before marking this layer done.
4. Once the profile is merged, regenerate and discard any hand-written code.

Until the gap is resolved, this layer is incomplete.
