* When to use this skill

When the user asks to run a named procedure — "run the X runbook",
"open a new sprint", "handle the PR review" — or any time you need
to execute a well-defined multi-step workflow end-to-end.

* How to use this skill

1. /Open the [[id:D1125E49-3C4E-48D4-B7F9-15CAA45B3ADD][Runbooks catalogue]]/ (=doc/llm/runbooks/runbooks.org=).
   Read the title and description of each entry to identify the
   closest match to the user's request. If no match is clear,
   show the user the catalogue titles and ask them to choose.

2. /Navigate to the matched runbook file/ via its id-link in the
   catalogue. Open =runbook.org= inside that folder.

3. /Execute every step in order/, respecting the runbook's
   preconditions, steps, and postconditions exactly. Do not skip
   or reorder steps.

4. /Confirm postconditions/ once all steps are complete and report
   the result to the user.
