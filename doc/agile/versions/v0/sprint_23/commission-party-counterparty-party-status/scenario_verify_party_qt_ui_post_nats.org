=PartyIdentifierDetailDialog= opens with the =id= field disabled (it's

new row appears in the parent's Identifiers table immediately (no
manual reload needed). Double-click the row to reopen it for edit;
confirm the =id= is shown but disabled, change the value, save, confirm
it round-trips. Select the row and click *Delete Identifier*; confirm
