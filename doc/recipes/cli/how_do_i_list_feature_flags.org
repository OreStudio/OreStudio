List feature flags via =ores.cli=. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I list feature flags?

* Answer

** List as JSON

#+begin_src sh :wrap src json
export ORES_CLI_DB_PASSWORD
./ores.cli variability feature-flags list ${db_args} ${log_args} --format json | jq .
#+end_src

** List specific entity

#+begin_src sh :wrap src json
export ORES_CLI_DB_PASSWORD
./ores.cli variability feature-flags list ${db_args} ${log_args} \
  --format json --key "system.bootstrap_mode" | jq .
#+end_src
