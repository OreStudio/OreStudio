* Summary

Two templates: the faker-based test-data generator for an entity's
value type — declaration and faker-cxx implementation — one synthetic
instance per call. =generator= profile.

* The ores.cpp.generator facet

Sibling to [[id:62342007-5080-43BF-8207-CD05B6568BBE][ores.cpp.domain]]: produces synthetic instances of the
domain class for tests. Column loops dispatch on per-field type flags
to choose the matching faker-cxx call and any generation constraints.
