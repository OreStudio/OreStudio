* Goal

Apply the =publish_from_dq= facet to
=ores.synthetic.market_data_generation_config= and its FX spot
sub-config so that a synthetic FX config becomes publishable from a DQ
bundle: the artefact-type row, shadow table, publish function
(=ores_synthetic_publish_fx_spot_config_from_dq_fn= or equivalent),
NATS handler, and registrar wiring should all be facet-generated —
this is the story's proof that the generic infrastructure holds for a
type outside =ores.refdata=/=ores.reporting=.
