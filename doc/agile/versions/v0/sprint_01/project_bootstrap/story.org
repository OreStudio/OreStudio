* Goal

By the end of this story, a contributor can clone the repo, build it
locally, and see a green CI pipeline produce a downloadable package. This
is the foundation every later story in v0 stands on.

* Acceptance

- A public GitHub repository exists for the project.
- =vcpkg= and =cmake= produce a clean local build.
- GitHub Actions run on every push, with at least one green pipeline.
- A built artefact is uploaded as a GitHub release/package.

* Decisions

- /Use vcpkg, not Conan/ :: chosen for its first-class CMake integration
  and the breadth of available ports.
- /Use GitHub Actions, not a self-hosted runner/ :: lower operational
  cost for a side project; trade hosted concurrency limits for zero
  maintenance.
- /Package as a GitHub release artefact/ :: avoids depending on an
  external package registry while the project's surface is still
  changing.

* Out of scope

- Code signing.
