* Goal

Raise the Qt UI from "functional" to "looks like a product":
project-specific branding (splash, app icon), modernised icon set
(Fluent), sensible window defaults.

* Acceptance

- Project-branded splash and neutral MDI background.
- Fluent icon set adopted.
- ORE Studio application icon in place.
- Windows don't auto-maximise; existing window reused on repeat
  invocation; the About menu describes the project.
