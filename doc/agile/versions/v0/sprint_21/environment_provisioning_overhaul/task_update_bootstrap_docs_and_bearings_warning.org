* Goal

Update the bootstrap developer environment runbook to reflect the genesis-env model (clone into
=ores_dev_prime_origin=, then provision worktrees via =compass env provision=). Add a three-way
detection block to =compass bearings= that warns when no genesis env exists at the conventional
sibling path, guides a fresh-clone user to move the checkout, and stays silent when the current
checkout IS the genesis env.
