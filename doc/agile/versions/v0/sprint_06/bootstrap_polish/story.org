* Goal

Polish the bootstrap surface introduced in sprint 05: move the
runtime context to the database, remove the bootstrap-mode flag from
context now that it isn't needed at runtime, and add manifest-level
password-validation control.

- Context carries no bootstrap flag.
- =system.disable_password_validation= readable from the manifest.
