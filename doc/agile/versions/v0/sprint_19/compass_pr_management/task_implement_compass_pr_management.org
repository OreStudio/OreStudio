* Goal

Implement the three review-round verbs as =compass review=
subcommands wrapping the =gh= CLI: =list= (all review comments with
id/file/line/preview, =--detail= for full bodies), =reply= (POST a
reply to a comment id with correct integer typing), and =resolve=
(resolve all unresolved threads via the GraphQL mutation, with
=--dry-run=). Defaults for =--owner=/=--repo= come from the git
remote. New module =compass_review.py=, dispatched from =compass.py=
following the existing short-circuit pattern.
