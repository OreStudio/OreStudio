* Goal

Add a test that walks every =.org= document in the repo, parses each
with =ores.orgmode=, and dumps the resulting structure to JSON. This is
a scale/robustness check against the real corpus (nothing should crash
or hang the parser on any doc actually in the repo, not just curated
fixtures) and, as a byproduct, gives a debuggable, diffable view of
what the parser extracts from any given doc — useful on its own while
developing or debugging the parser, independent of the test's
pass/fail result.
