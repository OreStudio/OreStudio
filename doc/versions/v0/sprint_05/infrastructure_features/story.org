* Goal

Two small but visible infrastructure features: a /detached/ default
for new MDI sub-windows, and a proper database-configuration mechanism
across console and Qt.

* Acceptance

- Detached vs attached default selectable as a setting; persists
  across sessions.
- Database configuration unified: CLI flags + env vars + config file,
  same mechanism in console and Qt UI.
