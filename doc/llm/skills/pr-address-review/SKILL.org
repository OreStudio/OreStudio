* When to use this skill

Review comments arrived on a PR: handle one complete round — sync, read, decide, fix, push, reply, resolve, record.

* How to use this skill

1. /Sync with main first/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh pr sync
   #+end_src


2. /Check CI/ before reading comments — a failure may supersede them:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh pr checks <N>
   #+end_src


3. /Read every comment/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh review list <N> --detail
   #+end_src


4. /Decide per comment/ — explicit accept or decline; never silently ignore.

5. /Fix/ — one commit per logical fix; never amend on a branch under review. Push.

6. /Reply to every comment/ — accepted: "Fixed in <sha> — ..."; declined: "Not changed. <reason>":

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh review reply <N> <comment-id> "<message>"
   #+end_src


7. /Resolve threads and record the round/ in the task's =* Review= table:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh review resolve <N>
   #+end_src
