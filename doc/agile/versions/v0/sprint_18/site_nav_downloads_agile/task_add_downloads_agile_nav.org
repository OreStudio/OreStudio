* Acceptance

- =doc/downloads.org= exists with =#+type: knowledge= frontmatter and a
  table whose columns are:
  =| Version | Code Name | Linux | macOS | Windows | Comments |=.
- Each cell in the Linux/macOS/Windows columns is either a hyperlink
  =[[url][Download]]= or =—= if no asset was published for that platform.
- The Comments column records notable absences or caveats (e.g. "No
  Windows build in this release").
- All releases with assets are included (see Notes for the full list).
- The latest release row is at the top.
- The page includes an alpha-software warning before the table,
  noting that the software is in early development, many problems are
  expected, and directing readers to the [[id:6A1A333C-A83A-8954-B9C3-C3AB1AEC3046][User Manual]] for guidance.
- =.build-site.el= =site-html-preamble= gains, in order after =Manuals=:
  #+begin_example
  <a href='/OreStudio/doc/downloads.html'>Downloads</a>
  <a href='/OreStudio/doc/agile/agile.html'>Agile</a>
  #+end_example
  These sit between the existing =Manuals= and =Roadmap= entries.
