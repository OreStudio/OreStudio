* Goal

Three infrastructure features that came along: an in-process event
bus (basis for upcoming Postgres-listener notifications), system tray
support for long-running sessions, and a past-timepoint faker for
tests.

* Acceptance

- Event bus exposes publish + subscribe; at least one real consumer.
- System tray icon with minimise-to-tray on supported platforms.
- Faker generator produces realistic past timepoints used by
  bitemporal tests.
