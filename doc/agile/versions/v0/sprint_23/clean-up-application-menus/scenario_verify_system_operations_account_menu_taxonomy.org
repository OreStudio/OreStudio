** Operations gains Telemetry, Message Queue, Configuration, User Accounts

Open *Operations*. Confirm it now contains, alongside the existing
Jobs and Workflows sections: *Configuration* (System Settings),
*Message Queue* (Queue Monitor), *Telemetry* (Log Viewer, Event
Viewer, Shell, Protocol, Settings — plus Service Dashboard as the
first item), and *User &Accounts* (Accounts, Roles, Tenants, Tenant
Types, Onboard Tenant...). Open one entry from each and confirm the
correct window/dialog opens. Confirm *User Accounts* is clearly
distinguishable from the personal *Account* actions now in File
(different labels: "User Accounts" vs "My Account").

** Data Management is retired; Classifications/Audit Trail moved to Data Quality

Confirm there is no more top-level *Data Management* menu. Open
*Data &Quality* and confirm it now contains, alongside the existing
Badges and Code Domains: a *Classifications* submenu (Coding
Schemes, Coding Scheme Authority Types) and an *Audit Trail*
submenu (Change Reason Categories, Change Reasons). Open one entry
from each and confirm the correct window opens.
