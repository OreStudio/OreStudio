* Acceptance

- Pattern: {platform}-{compiler}-{buildtype}-{tool}.
- Hidden ninja + make presets centralise generator settings.
- Linux + macOS support both; Windows on Ninja only.
- Automatic parallel-build detection removed.
- Docs + skill files updated.

* Tasks

In execution order:

- [[id:a77a88d5-737a-4136-85fa-f89e013d4908][Add build-tool suffix to CMake presets]]

* Decisions

- /Explicit suffix everywhere/ :: even if there's only one tool for a platform, the suffix makes scripts portable.
- /Defer parallelism to env / tool default/ :: removes one moving part from CMakeLists.txt.

* Out of scope

- Adding a third build tool — sticking with the two.
