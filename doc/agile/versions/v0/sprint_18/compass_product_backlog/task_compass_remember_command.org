* Goal

Add a =compass remember= subcommand that scaffolds a new project memory
file and registers it in =doc/llm/memory/memory.org=. Interface:

#+begin_example
compass remember --what "Rule or fact to remember" --why "Reason / incident"
#+end_example

The command generates a =.org= memory file (via =generate_v2_doc.sh=
with =--type memory=), fills in the body from =--what= and =--why=,
and appends a catalogue entry to =doc/llm/memory/memory.org=.

* Acceptance

- =compass remember --what "X" --why "Y"= creates a new memory file
  under =doc/llm/memory/= with correct frontmatter and body.
- Body contains the rule/fact from =--what= followed by a =*Why:*= line
  from =--why= and a placeholder =*How to apply:*= line.
- The memory is appended to the =** Feedback= section of
  =doc/llm/memory/memory.org= with an org-roam link.
- =--subtype= flag (default: feedback) controls which catalogue section
  the entry goes into (feedback / user / project / reference).
- Running the command twice with the same slug errors rather than
  creating a duplicate.
