* Goal

CI cost and signal scale with tens of merges an hour: pull requests
run only the jobs their changed paths warrant behind a single
required check, main never races itself, and full platform matrices
run on staggered schedules that skip when nothing relevant changed.
Reviews keep working after Gemini's July 2026 sunset.

* Acceptance

- A doc-only PR runs the site build (plus second-scale lint checks)
  and nothing else; a code PR runs canary and roundtrip and no site
  build; a CI-config PR runs everything.
- pr-gate is the only required check and always reports — doc-only
  PRs merge without --admin.
- No compile workflow triggers on push to main; Linux runs every 2h,
  macOS every 3h, Windows every 4h, each skipping when only doc-class
  paths changed since its last green run; coverage rides the Linux
  debug leg as today.
- Site deploy remains per-merge and coalesced.
- Claude reviews PRs on open; compass review verbs work against its
  comments.

* Plan

Brainstormed and validated 2026-06-06.

- /PR tier/: one =pr.yml= with a path-classifier first job (classes:
  docs =doc/**, assets/**, .claude/**=; code =projects/**=, CMake,
  vcpkg — including =projects/**/modeling/*.org= codegen models; ci
  =.github/**= and codegen templates). Jobs run per class (docs →
  site; code → canary + roundtrip; ci → everything; mixed → union);
  a final =pr-gate= job is the single required check. Chosen over
  per-workflow =paths:= filters because path-filtered required
  checks hang as "expected" — the cause of today's --admin
  workaround on doc-only PRs.
- /Main per merge/: only the coalesced site deploy. All compile
  triggers removed from push.
- /Main scheduled/: Linux every 2h, macOS every 3h, Windows every 4h
  (staggered by build cost), =cancel-in-progress: false=, each
  prefixed by a relevance guard that diffs HEAD against the
  workflow's last successful run SHA and skips when only doc-class
  paths changed. Coverage needs no move: it rides the Linux debug
  leg (=code_coverage=1=); =ore_coverage.yml= is actually the
  roundtrip check and folds into the PR code class.
- /Observability/: CDash + existing Discord; no new notification
  machinery.
- /Reviews/: =anthropics/claude-code-action@v1= on PR open/sync with
  ANTHROPIC_API_KEY and a correctness prompt replaces Gemini (sunset
  July 2026).
- Rejected: coalescing latest-only main builds (constant burn under
  sustained merging); GitHub merge queue (strong but heavy for a
  single-maintainer fleet); per-workflow path filters (required-check
  hang).
