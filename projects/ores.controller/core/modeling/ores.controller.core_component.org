#+brief: Core lifecycle management logic for the ORE Studio service controller.

Contains the process supervisor, replica manager, restart policy engine,
and NATS message handlers for the service controller. Reads service
definitions from the database, spawns and monitors OS processes, writes
service instance state and lifecycle events, and handles NATS commands
for starting, stopping, and restarting individual service replicas.
