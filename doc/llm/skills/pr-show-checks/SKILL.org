* When to use this skill

Inspecting CI state on a PR: which checks pass, fail, or are pending.

* How to use this skill

1. /Show checks/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh pr checks <N>
   #+end_src

   Add =--watch= to poll until completion.

2. /On failure/, follow [[id:62D39F43-5D91-4A82-B1C3-62C2AF21BEFC][How do I fix a failing CI check?]] — fix as a separate commit before anything else.
