* Answer

** List all as JSON

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli iam roles list ${db_args} ${log_args} --format json | jq .[:3]
#+end_src

** List specific role

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli iam roles list ${db_args} ${log_args} \
  --format json --key "Admin" | jq .
#+end_src
