#+title: task start crashes after switching to a stale branch
#+description: compass task start resolves the task doc path, switches to the task's branch, then re-reads the doc — but on a branch that diverged before a sprint transition the file lives elsewhere (or compass itself is older), producing FileNotFoundError tracebacks and an unstamped journal. Re-resolve the doc after the switch, or read everything needed before switching; also consider warning when the target branch's compass predates the running one.
