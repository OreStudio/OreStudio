** Proposed entry format

#+begin_example
* 2026-05-31 09:15 — [[id:3B8F2A1D-7C4E-4B6A-9E5D-2F1A8C3B7D9E][Refactor ores.codegen C++ generation]]
  - Task :: [[id:D4469217-12E2-41AC-AB35-AC70E849D88C][Codegen architecture analysis and unified model roadmap]]
  - State :: DONE
  - Branch :: feature/codegen-architecture-analysis
  - PR :: #945
#+end_example

Fields:
- *Timestamp* — ISO date + time; written by =compass journal update= at call time.
- *Story link* — org-roam =[[id:...][Title]]= for the parent story.
- *Task link* — org-roam =[[id:...][Title]]= for the specific task.
- *State* — task state at time of update (STARTED, DONE, etc.).
- *Branch* — current git branch name.
- *PR* — GitHub PR number, or =none= if no PR open yet.

** Parser notes

=compass journal where= reads entries by scanning for =^* \d{4}-= lines
and parsing the most recent block. =compass fleet= reads the last entry
from each worktree's =.journal.org= using the same parser.
