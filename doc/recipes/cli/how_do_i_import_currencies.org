Import currencies into the database via =ores.cli= from ORE XML files. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I import currencies?

* Answer

** Import single file

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli refdata currencies import ${db_args} ${log_args} \
  --target ${data_dir}/currencies_API.xml
#+end_src

#+RESULTS:
#+begin_src json
"currencies_API.xml": Imported a total of 178 currencies.
#+end_src

** Import multiple files

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli refdata currencies import ${db_args} ${log_args} \
  --target ${data_dir}/currencies_API.xml \
  --target ${data_dir}/currencies_42.xml
#+end_src

#+RESULTS:
#+begin_src json
"currencies_API.xml": Imported a total of 178 currencies.
"currencies_42.xml": Imported a total of 2 currencies.
#+end_src
