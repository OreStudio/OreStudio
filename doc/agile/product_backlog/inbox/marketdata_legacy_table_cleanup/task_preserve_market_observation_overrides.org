* Goal

Stop =compass codegen regenerate --component marketdata --profile
all-cpp= from silently reverting =market_observation='s hand-written
behaviour: a =series_id= filter parameter on the paginated list
query (repository + service + NATS protocol) and custom pagination
logic in =market_observation_repository::read_latest=. Discovered
while regenerating for [[id:82CB5F03-AFD9-4AE6-95EF-57EB683FB1B5][Hotfix: market_fixing entity carries audit
columns absent from lean SQL schema]] — every regen pass had to be
followed by a manual =git checkout HEAD --= on these files. Either
model the =series_id= filter properly in
=ores.marketdata.market_observation.org= so codegen produces it
natively, or clearly mark the hand-maintained blocks (e.g. a
=// CODEGEN: preserve= comment convention, if one exists, or a new
one) so future regenerations don't clobber them.
