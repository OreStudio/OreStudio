* Goal

Detail dialogs were growing long and inconsistent. Convert them all
to a tabbed layout with a dedicated Provenance tab, then drop the
in-dialog toolbars in favour of plain buttons.

* Acceptance

- All detail dialogs tabbed with consistent Provenance handling.
- Toolbars removed; delete / cancel / save buttons standard.
- My Sessions + My Account land as MDI subwindows.

* Tasks

In execution order:

- [[id:ba771ea9-fbf1-420e-b708-e12a3276758f][Make all detail dialogs tabbed]]
- [[id:6b0d2950-4144-4c15-b4fa-356b6ec0770b][Remove toolbars from detail dialogs]]

* Decisions

- /Provenance tab always last/ :: provenance is meta-data; doesn't belong in the main user flow but should always be there.
- /Plain buttons over toolbars/ :: the host MDI window already provides the toolbar; duplicating it in the dialog was clutter.

- Theming the dialog headers.
