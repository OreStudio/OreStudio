* Goal

Settle the design questions before any mass migration: how the 140
templates map onto the three-level literate hierarchy (template ←
facet doc ← group doc ← groups overview) — what counts as a facet
group, e.g. =cpp_domain_type_*= — where the org sources live, how
tangling runs (org-babel via
the existing =ores-babel.el= / tangle tooling), what the generated-file
header looks like inside a =.mustache= artefact, and how the zero-diff
drift check is wired (CI target and/or compass command). Prove the
design with a small pilot batch of templates and record the decisions
in the parent story's =* Decisions= section.
