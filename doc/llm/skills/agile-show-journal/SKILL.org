* When to use this skill

Reviewing where we have been: the per-worktree session journal of story and task journeys.

* How to use this skill

1. /Latest entry/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh journal where
   #+end_src


2. /Full log/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh journal log
   #+end_src

   Writes happen automatically as side-effects of start, done, and PR skills — never write the journal by hand.
