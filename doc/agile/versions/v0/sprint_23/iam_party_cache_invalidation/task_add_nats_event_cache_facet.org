* Goal

Add a =nats-event-cache= codegen facet generating two archetypes for
any tenant-scoped entity that opts in: the consumer-side
={entity}_cache.hpp= (a =partitioned_cache= instantiation warmed and
reloaded via the entity's =read_for_cache= RPC) and a
={entity}_cache_registrar.hpp= snippet (startup warm-up + change-event
subscription, subject-bound via =event_traits=, no hardcoded strings).
Declared via =cached_by: <component>= on the entity's own model, whose
output lands in the *consumer's* component tree — not the entity's own
— since the generated cache must compile into the consumer's binary
(e.g. party is defined in =ores.refdata= but its cache would compile
into =ores.iam=).

This task builds and verifies the facet only; it does not wire party
onto it (that is the next task, "Migrate party to the generated
cache").

* Acceptance

- A new =ores.cpp.nats-event-cache= facet exists, disabled by default;
  an entity opts in via =:ores.cpp.nats-event-cache.enabled: true=
  (file-level drawer) plus =cached_by: <component>= and
  =read_for_cache: true= (both in =** Flags=).
- Opting in generates, with no hand-written code:
  - ={entity}_cache.hpp= — a =partitioned_cache<std::string, PK, Value>=
    instantiation with =load(tenant_id)= (calls =read_for_cache=,
    replaces the tenant's partition) and =lookup(tenant_id, key)=.
  - ={entity}_cache_registrar.hpp= — =warm_and_subscribe_{entity}_cache()=
    doing startup warm-up + =event_traits=-bound reload-on-change.
  - Both land under =projects/ores.<cached_by>/core/...=, not the
    entity's own component tree.
- =cached_by= requires =read_for_cache= (validated, raises otherwise);
  entities that don't opt in are unaffected (regression-checked).
- Facet/archetype docs describe the new flags and mechanism.
- party's model/messaging layer is untouched by this task.

* Result

Added the =ores.cpp.nats-event-cache= codegen facet, disabled by
default and opt-in per entity:

- =core.py=: =resolve_output_path= gains ={cache_component}=/

    =cpp_nats_event_cache.hpp.mustache=): ={entity}_cache.hpp=, a
