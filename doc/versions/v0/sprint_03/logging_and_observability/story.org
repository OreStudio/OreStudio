This page documents a [[id:699a8d45-b67e-4d3e-9206-24fa17c51ada][story]] in *Sprint 03*. It captures the goal, current status, acceptance criteria, and the tasks that compose it.

* Goal

Get the logging path right (proper Boost.Log severity channels and
sinks rather than ad-hoc =std::cerr=), triage the Valgrind leaks
surfaced earlier, and lift code coverage in the under-tested
modules.

* Decisions

- /Boost.Log channels for component-level filtering/ :: lets a
  reader filter logs by component without parsing free-form prefixes.
- /Suppress upstream Valgrind noise rather than chase it/ :: the
  OpenSSL false positives don't represent real leaks; documented in
  the suppression file.

* Out of scope

- Replacing Boost.Log with another logging library.
- Distributed tracing (introduced later in a telemetry story).
