Export currencies via =ores.cli= to ORE XML, CSV, or JSON. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I export currencies?

* Answer

** Export all as ORE XML

#+begin_src sh :wrap src xml
export ORES_CLI_DB_PASSWORD
./ores.cli refdata currencies export ${db_args} ${log_args} --format xml | \
    xmllint --format - | \
    head -n 20
#+end_src
