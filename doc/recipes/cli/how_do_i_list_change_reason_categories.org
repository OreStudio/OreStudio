List change reason categories via =ores.cli=. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I list change reason categories?

* Answer

** List all as JSON

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli dq change-reason-categories list ${db_args} ${log_args} --format json | jq .
#+end_src

** List all as table

#+begin_src sh :wrap src org
export ORES_CLI_DB_PASSWORD
./ores.cli dq change-reason-categories list ${db_args} ${log_args} --format table
#+end_src

#+RESULTS:
#+begin_src org

+--------+-------------------------------------------------------------------------+-------------+---------+
| Code   | Description                                                             | Recorded By | Version |
+--------+-------------------------------------------------------------------------+-------------+---------+
| common | Universal data quality reasons aligned with BCBS 239 and FRTB standards | system      | 1       |
| system | System-generated reasons for automatic operations (not user-selectable) | system      | 1       |
| trade  | Trade lifecycle reasons aligned with FINRA and MiFID II standards       | system      | 1       |
+--------+-------------------------------------------------------------------------+-------------+---------+


#+end_src

** List specific category

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli dq change-reason-categories list ${db_args} ${log_args} \
  --format json --key "common" | jq .
#+end_src
