* Acceptance

- pgmq + pg_cron extensions removed.
- Custom ores_mq_* tables with scope + queue type + RLS.
- In-process scheduler_loop using boost::asio.
- TimescaleDB hypertable for job instance history.
- Protocol 48.0 (breaking).
