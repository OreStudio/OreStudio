* Acceptance

- =compass site serve= starts the HTTP server on =ORES_SITE_PORT= from =.env=.
- =compass site serve --compile= rebuilds the site first then serves.
- =compass site serve --port N= overrides the port.
- =build/scripts/serve-site.sh= is deleted from the repo.
- The "How do I serve the site locally?" recipe is updated to use =compass site serve=.
- =ores-dashboard.el= reference to =serve-site.sh= is updated.

* Decisions

- Used Python's =http.server.HTTPServer= (stdlib) rather than a third-party server — no new dependency.
- Port resolution order: =--port= flag → =ORES_SITE_PORT= from =.env= → =51004= (local1 default).
- =fuser -k= on the port before starting: mirrors the old script's =--kill= behaviour automatically.
- Buffer name in =ores-dashboard.el= kept as ="serve-site"= — only an Emacs buffer suffix, not a path reference.
