#+HTML: <h1>Enterprise-grade risk analytics &mdash;<br/>but visual <i>and</i> open-source.</h1>
#+HTML: <p>ORE Studio wraps the <a href="https://github.com/OpenSourceRisk">Open-Source Risk Engine</a> (ORE) in an intuitive graphical interface - no Python or C++ required.</p>
#+HTML: </section>

*ORE Studio* is a new and independent open-source project with the goal of
providing a graphical wrapper around Acadia’s [[https://github.com/OpenSourceRisk/Engine][ORE]]. ORE itself builds on top of
[[https://github.com/lballabio/QuantLib][QuantLib]], the de-facto standard open-source library for quantitative finance.
*ORE Studio* aims to augment ORE's robust analytics to eventually provide full
life-cycle management for all entities within the trading system,
point-and-click reporting, trade modelling and market-data management.
