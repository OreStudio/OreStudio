* Decisions

- Disabled =CMAKE_CXX_SCAN_FOR_MODULES= globally rather than scoping
  it to Windows-only presets: the project has no C++20 module usage
  anywhere, so the scan pass is dead weight on every platform, and a
  single global setting is simpler to maintain than a per-preset
  override. Revisit if/when the project adopts C++20 modules.
