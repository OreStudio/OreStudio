Recipes related to *codegen* — the model-driven C++ generator and the
documentation scaffold. Some entries below appear in both this
inventory and [[id:95FF6AF4-2320-4CB6-A007-B016B7DA16ED][Documentation recipes]] because they operate on docs
/through/ codegen; the file's path tells you where it physically
lives.

* Scaffolding documents (via codegen)

These recipes drive =generate_doc.sh= — the codegen-side wrapper
that produces documents. They also appear in [[id:95FF6AF4-2320-4CB6-A007-B016B7DA16ED][Documentation
recipes]] under "Authoring documents".

- [[id:3153AA4B-23D4-4111-9710-5A4971102EDC][How do I create a new doc?]] — generic flow for every doctype.
- [[id:0806C8FB-43D2-497E-900D-5F2E0B3472FD][How do I create a component overview?]] — component-specific flow.
- [[id:D4E65EF1-54F4-47E1-9388-58BE9E36FBE3][How do I create a recipe?]] — recipe-specific conventions and
  inventory wiring; physically under [[proj:doc/recipes/documentation][=recipes/documentation/=]].
- [[id:391B9231-DBA8-4E8C-97AB-A75E2B9FEA07][How do I create a memory?]] — memory-specific conventions;
  physically under [[proj:doc/recipes/documentation][=recipes/documentation/=]].

* Entity management (compass codegen entity)

These recipes cover the =compass codegen entity= sub-commands — entity-level
operations that resolve entities by name or UUID prefix without needing to
know model paths or component names.

- [[id:923AE3E7-5B55-43F4-967C-0E71AB9F2C46][How do I list codegen entities?]] — enumerate entities by metatype; get short UUID for =compass show=.
- [[id:BF893756-546E-4EAF-BB3C-C88BD16B3935][How do I show generated files for a codegen entity?]] — staleness icons and archetype template per output file.
- [[id:8010834F-9179-4ECD-999D-5D017D393D36][How do I generate a codegen entity?]] — regenerate an entity's full supported set; =--address= to restrict to one facet; =--dry-run= to preview paths.
- [[id:872903C8-1522-49EA-A378-647FA6F48C4F][How do I preview entity codegen changes as a diff?]] — generate to temp and show unified diff without writing.
- [[id:839B1943-34A9-443B-9483-CFC45D6A7644][How do I diff generated files for an entity?]] — git diff of on-disk generated files after generation.
- [[id:153D67F4-F420-4983-AA3F-0A3246477BEE][How do I discover the physical-space catalogue?]] — list every ores.* address, or an entity's supported set with per-archetype ✅/❌ status and exclusion reasons.
