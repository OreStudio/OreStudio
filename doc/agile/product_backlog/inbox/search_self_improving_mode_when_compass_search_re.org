#+title: Compass search: self-improving mode for failed queries
#+description: Search self-improving mode: when compass search returns no results or the user resorts to manual commands to find what they wanted, enter a self-improving mode. Track subsequent compass searches and shell commands used to compensate. Once the user locates the target document, auto-create a detailed capture recording the failed query, the workaround path taken, the found document, and a suggested improvement to the scoring/indexing. This corpus of failures becomes training data for tuning the search algorithm.
