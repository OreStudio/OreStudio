* Summary

Slovaris is a hand-crafted imaginary-world taxonomy used to seed the
=ores.refdata= schema for local development and CI. It provides 100
country/currency pairs drawn from the fictional =Aerilon/Ducat=
universe, plus the catalog, dataset, tag, flag, and methodology rows
that the refdata FK graph requires to be self-consistent.

* Why this dataset

The refdata schema has deep FK chains: currencies belong to countries;
countries and currencies belong to catalogs; catalogs belong to
methodologies. A synthetic row-generator would need to respect this
closure automatically. The slovaris dataset solves the problem with
hand-authored rows whose relationships are meaningful and stable —
each =Aerilon= country maps to exactly one =Ducat= currency, giving
test queries a predictable answer set.

The fictional framing also makes it immediately obvious in a log or
query result that a row is test data, not a production import.
