** Connect to tenant Barclays Plc

Log in against the =prime_origin= environment as
=tenant_admin@barclays_plc= / =Secure-Password-123= and select
*BARCLAYS PLC*. If the database has been recreated since last use,
re-provision first: =compass shell -f
projects/ores.shell/scripts/library/provisioning/barclays_system_provision.ores=
(publishes real FX driver-rate observations needed for this test).

** Open an FX Spot chart window

Menu: *Market Data > FX Spot* (or the equivalent currency-pair chart
window). Pick a currency pair that has data, e.g. EUR/USD.

** Confirm the chart renders observation data

Confirms PASS if the chart plots a non-empty price series for the
selected pair with no error dialog or empty-state message. Confirms
FAIL if the chart is blank, shows an error, or the app logs a failed
=marketdata.v1.market_observations.list_by_series_id= request.

** Confirm no console/log errors on load

Check the running client's log
(=build/output/linux-clang-debug-make/publish/log/ores.qt.<colour>.log=)
for the chart-loading window: confirms PASS if it shows a successful
=list_by_series_id= round trip (no =Failed to fetch observations=
message).
