* Goal

Port =dogen.org='s entity model and parser
(=~/Development/masd/dogen.local1/projects/dogen.org=) into
=ores.orgmode=, adapted to this project's current codegen/component
conventions (its getter/setter style predates them). Reuse the
document/headline/drawer/section/block shape as-is where it fits;
add what it's missing — a first-class =id:= link entity (target UUID +
display text) and a =#+type:= field surfaced on the document, since
=dogen.org= has no notion of either. Scope the parser to the subset of
org syntax this project's agile/knowledge docs actually use (frontmatter
=#+= keywords, =:PROPERTIES:= drawer, headings, plain paragraphs,
lists, tables, links) — not full org-mode.
