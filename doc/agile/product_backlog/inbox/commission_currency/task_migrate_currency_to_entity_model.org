- =currency.hpp=: codegen missing standard audit fields (=version=, =tenant_id=,
  =modified_by=, =change_reason_code=, =change_commentary=, =performed_by=, =recorded_at=);
  generates =created_at= instead of =recorded_at=. These are template gaps to resolve in
  the sync C++ domain task.
- =ClientCurrencyModel.hpp=: codegen generates a simpler model class without =kColumns=
  metadata array, =kDefaultWindowSize=, or =ImageCache= support. Qt sync task will
  classify which sections are template improvements vs custom code.
- =currency_generator.hpp=: codegen generates a simpler interface; custom generator
  variants (fictional, unicode, unique) are injected via the entity model's custom
  generator sections.
- =currency_repository.hpp= / =.cpp=: codegen base template missing per-key overloads and
  paginated read; custom repository sections in entity model inject these.
- =currency_service.hpp= / =.cpp=: custom service methods (party-filtered lists)
  injected via entity model; confirmed present in generated output.

=currency_table.org= remains in place; the decommission story (69A50205) tracks its
eventual removal once the sync tasks are complete and the new entity model is live.
