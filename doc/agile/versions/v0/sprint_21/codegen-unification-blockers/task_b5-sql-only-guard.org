* Goal

A domain_entity model with #+sql_only: true in its frontmatter does not dispatch any C++ or Qt profile. The flag is read from frontmatter by load_org_model() and surfaced in the model dict as domain_entity.sql_only. Profile dispatch in core.py checks this flag before firing non-SQL profiles.
