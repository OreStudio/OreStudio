* Decisions

- /Wt over a JS frontend/ :: lets us share the same C++ domain types between desktop and web; one source of truth.
- /HTTP reuses socket handlers/ :: enabled by the type-traits dispatch from the comms story; otherwise we'd have two copies of every handler.
