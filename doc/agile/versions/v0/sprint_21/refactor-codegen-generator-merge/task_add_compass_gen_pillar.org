* Goal

Expose compass gen generate and compass gen regenerate subcommands that delegate to the ores.codegen CLI. Compass becomes the single documented entry point for code generation. The generate pillar adds compass gen generate (single model) and compass gen regenerate (all models for a component). Wire any existing callers (recipes, skills, runbooks) to use the compass gen verbs instead of calling codegen.sh directly.
