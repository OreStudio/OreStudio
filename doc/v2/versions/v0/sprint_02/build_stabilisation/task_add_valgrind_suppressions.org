* Goal

The Valgrind nightly was producing enough noise from third-party
libraries that real issues were being missed. Add a suppression file
covering the known false positives.

* Acceptance

- A Valgrind suppression file is committed and used by the nightly.
- Each suppression entry has a comment explaining the cause.
- The nightly output is meaningfully shorter.

* Notes

- Suppressions are kept narrow (specific symbols / libraries) to
  avoid masking real bugs.

* Result

- Suppression file in place; nightly noise reduced to a useful
  signal-to-noise ratio.
