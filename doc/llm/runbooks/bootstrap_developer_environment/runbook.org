* Goal

Set up a fresh ORE Studio development environment from zero: clone, install dependencies, configure CMake, build, run tests, and verify.

* Preconditions

- Git, CMake, Ninja, and a supported C++23 compiler are installed.
- PostgreSQL is available (local or container).

* Steps

In execution order:

1. /Clone the repository./ =git clone https://github.com/OreStudio/OreStudio.git= and =cd= into it.

2. /Install dependencies./ Follow [[id:49F7DA81-ADB7-44ED-BE7E-06DBFBB32A84][How do I set up a development environment?]] for vcpkg bootstrap and system packages.

3. /Configure CMake./ Follow [[id:7C44751E-54C0-4CB7-B489-1D3EC609A4E3][How do I configure the project?]]: =cmake --preset linux-clang-debug-ninja=.

4. /Build./ Follow [[id:F176FCA1-68D5-420D-B076-75A65FEE5EBB][How do I build the system?]]: =cmake --build --preset linux-clang-debug-ninja=.

5. /Run tests./ Follow [[id:20D2CB34-4E2F-45D3-8FA2-F1C5B4C7B22F][How do I run the tests?]]: =cmake --build ... --target rat=.

6. /Verify./ Run =ores.cli --help= or launch the Qt desktop client to confirm a working end-to-end setup.

* Postconditions

- All build targets compile.
- All tests pass.
- CLI and Qt binaries are functional.
