* Goal

Ensure the C++ core codegen templates (=domain=, =repository=, =service=,
=generator= profiles) produce output that matches the repository for
=country=. Fix templates to match code — not the reverse — unless the
code is genuinely wrong.

These profiles generate:
- =domain=: domain struct (=country.hpp=), SQL-mapping table (=country_table.hpp/.cpp=), JSON IO (=country_table_io.hpp/.cpp=)
- =repository=: entity (=country_entity.hpp/.cpp=), mapper (=country_mapper.hpp/.cpp=), repository (=country_repository.hpp/.cpp=)
- =service=: service class (=country_service.hpp/.cpp=)
- =generator=: test data generator (=country_generator.hpp/.cpp=)

All C++ artefacts for country are currently hand-written (no domain entity
org model exists). This task creates the domain entity org model and
iterates until the profiles produce zero diff.
