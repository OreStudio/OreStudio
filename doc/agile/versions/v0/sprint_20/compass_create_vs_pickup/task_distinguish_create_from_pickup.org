* Goal

Make the create vs pick-up distinction first-class in compass.
Creating a story or task (=compass add story= / =add task=) scaffolds
documents only. Picking one up (=compass task start=) creates the
branch, marks STARTED, and stamps the journal — and works on any task
without manual branch plumbing. =compass story new= becomes an honest
pick-up composite: the scaffolding work rides its own scaffold task,
which owns the branch and which the scaffold PR closes; the first
real task is born BACKLOG with no branch.
