List login info entries via =ores.cli=. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I list login info entries?

* Answer

** List all as JSON

#+begin_src sh :wrap src json
export ORES_CLI_DB_PASSWORD
./ores.cli iam login-info list ${db_args} ${log_args} --format json | jq .
#+end_src

#+RESULTS:
#+begin_src json
[
  {
    "last_login": "2025-12-12 16:07:21.000000000Z",
    "account_id": "e7bddd23-c004-4652-99e1-8d4ff5856fe1",
    "failed_logins": 0,
    "locked": false,
    "online": false,
    "last_ip": "0.0.0.0",
    "last_attempt_ip": "0.0.0.0"
  }
]
#+end_src

** List all as table

#+begin_src sh :wrap src org
export ORES_CLI_DB_PASSWORD
./ores.cli iam login-info list ${db_args} ${log_args} --format table
#+end_src

** List specific feature flag

#+begin_src sh :wrap src json
export ORES_CLI_DB_PASSWORD

  --format json --key "019b1008-6da1-733c-a6e6-e942eaab2ed6" | jq .
