Per-entity NATS sub-registrar implementation. Defines
=register_<entity>_handlers()= which creates the handler and wires its
NATS subjects. =nats-sub-registrar= profile.

Each entity's registrar includes only its own handler header, so a compile
failure in one entity's handler cannot block the rest of the service.

Model flags consumed by this template:

- =no_history= (bool, Flags) — skip the history subject subscription.
  Set on entities whose service has no =get_<entity>_history= method.
- =list_only= (bool, Flags) — wire only the list subject.
  Set on read-only lookup entities.

variables and their semantics.
