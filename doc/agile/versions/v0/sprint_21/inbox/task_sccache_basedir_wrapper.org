* Goal

Without SCCACHE_BASEDIR each worktree compiles every TU from scratch (100% cache misses) because the worktree-specific path prefix differs in all =-I= flags. Add a committed wrapper script that sets SCCACHE_BASEDIR to the worktree root at build time so all worktrees on the same commit share a single sccache entry per TU.

* Result

Merged as PR #1294. `build/scripts/sccache_wrapper.sh` committed to git; resolves project root at runtime from `build/scripts/../..` so it works in any worktree. `CMakeLists.txt` uses the wrapper as `CMAKE_C/CXX_COMPILER_LAUNCHER` with `FORCE`. After each worktree runs `cmake --preset`, sccache entries built in one worktree are reused by all others on the same commit.
