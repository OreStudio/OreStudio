* Question

How do I add a new change reason category using =ores.cli=?

* Answer

#+begin_src sh :wrap src text
export ORES_CLI_DB_PASSWORD
./ores.cli dq change-reason-categories add ${db_args} ${log_args} \
  --code "test_category" \
  --description "Test category for CLI demonstration" \
  --recorded-by admin \
  --change-commentary "Adding test category via CLI"
#+end_src
