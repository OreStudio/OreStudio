* Summary

This document is the single source of truth for the =.env= /format
version/ — the integer written as =ORES_ENV_VERSION= in every generated
=.env=. The *current version is the highest row* in the table below.
=compass env init= reads it and stamps it into the =.env=; the version
check (run before =recreate_database=) compares a checkout's
=ORES_ENV_VERSION= against it and tells the user to re-run
=compass env init= if the =.env= is older. Record a new version with
=compass env new-version "<description>"=, which appends a dated row —
do this whenever a change alters the =.env= shape (a new service in the
[[id:CB7713BA-544A-43CD-AEB3-30C2157CBBCC][service registry]], a new or renamed variable).

* Versions

The table is machine-parsed: each data row is =| <integer> | <date> |
<description> |=. Do not edit it by hand — use =compass env new-version=.
