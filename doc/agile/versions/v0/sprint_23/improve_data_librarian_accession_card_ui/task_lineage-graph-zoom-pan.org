* Goal

The Dependencies tab's =QGraphicsView= lineage graph (=DatasetViewDialog::updateLineageView=)
only ever shows the initial =fitInView= framing — there's no way to
zoom in on a dense graph or pan around one that doesn't fit. Add
mouse-wheel zoom and drag-to-pan, matching how other interactive
graph/canvas views in the app behave. Surfaced during manual QA of
the Phase 2 two-column layout.

- Mouse wheel zooms the lineage graph in/out, centered on the cursor.

  original =fitInView= framing.
- Manually verified against a dataset with several dependencies.
