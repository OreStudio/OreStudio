* Goal

=compass add lookup_entity= should be as safe against silent
knob-omission as =compass add entity_org --shape= now is: parameterize
=has_coding_scheme=/=has_image_id=/=has_artefact_insert_fn= via
=--entity-<knob>= flags instead of the current hardcoded =false=, and
extend the scaffold's =Primary key= section to support a compound
natural key (more than one column), so [[id:819B050A-A877-4B9A-BB24-942B95DA1206][the sibling DQ commissioning
task]] can model =subject_area= (keyed by =name= + =domain_name=)
accurately instead of misrepresenting its uniqueness constraint.
