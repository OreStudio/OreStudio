* Goal

Stand up an MQ subsystem on top of pgmq with a full Qt monitoring UI.

* Acceptance

- MQ subsystem protocol messages 0xB000-0xB013 for queue management + messaging (create / delete / purge / send / read / pop / delete).
- =ores_mq_metrics_samples_tbl= TimescaleDB hypertable for queue metrics.
- =ores_mq_scrape_metrics_fn= pg_cron job scrapes + persists metrics every minute (initialised on service startup).
- Qt monitor UI: =ClientQueueModel= (live merged view of metadata + metrics), =QueueMonitorMdiWindow= (queue CRUD + charting), =QueueDetailDialog= (publish + interact with messages), =QueueChartWindow= (time-series charts).
- Protocol 46.3.
