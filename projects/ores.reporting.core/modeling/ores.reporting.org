Reporting infrastructure for [[id:D773166D-0C91-8CB4-3323-42166BC07687][ORE Studio]]. Manages report types,
report definitions, report instances, and concurrency policies for
executing ORE risk report runs.

* Facets

The component is organised into the following facets:

| Facet | Description |
|-------+-------------|
| domain | Report type, definition, instance, and concurrency policy types |
| service | Business logic for report management and execution |
| repository | Persistence layer for reports and run history |
| messaging | NATS request/reply handlers — see [[id:B1A5C952-B709-444B-8A63-5FD834D87D52][ores.reporting Messaging Reference]] |
