* When to use this skill

When the user asks to create a new Claude Code skill for ORE Studio,
slim an existing skill into the v2 thin-index shape, or rebuild the
shipped skill bundle.

* How to use this skill

1. Confirm the work is /skill-shaped/ — a thin index over recipes and
   a knowledge document, not the recipes themselves. See =skill= in
   [[id:8b29a8d8-e004-4e87-b81d-abeaf68f2b98][document_types]] for the contract.
2. Identify the recipes the skill will index. Each operational
   how-to belongs in its own =how_do_i_<thing>.org= recipe (see
   [[id:3153aa4b-23d4-4111-9710-5a4971102edc][How do I create a new v2 doc?]]). If a recipe doesn't exist yet,
   create it first using =--type recipe=.
3. Generate the skill scaffold with the v2 doc generator
   (=--type skill=). The slug uses =kebab-case= and doubles as the
   Claude Code =name:= field. See [[id:3153aa4b-23d4-4111-9710-5a4971102edc][How do I create a new v2 doc?]] for
   the exact invocation.
4. Fill in the skeleton:
   - =* When to use this skill= — one or two sentences.
   - =* How to use this skill= — numbered steps.
   - =* Recipes= — id-link to every related recipe.
   - =* Reference= — id-link to the knowledge document and the
     component model that back the skill.
5. Rebuild the shipped bundle to verify the skill exports cleanly —
   see [[id:b5f231a9-5403-482c-8f2a-12d6917c58cd][How do I deploy the skills?]].
