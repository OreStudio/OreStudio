** HistoryDialogBase hook delegation

Template-generated history dialogs fully delegate to =HistoryDialogBase=:

- Constructor calls =initializeHistoryUi({versionListWidget, changesTableWidget, titleLabel, closeButton})= after setting column headers; all toolbar, selection, and close-button wiring is handled by the base.
- =loadHistory()= uses =runHistoryRequest<>()=; the =on_success= lambda stores response data and calls =historyLoaded()=.
- Overrides: =historySize()=, =versionRow()=, =historyTitle()=, =calculateDiffAt()=, =displayFullDetails()=, =openVersionAt()=, =revertToVersionAt()=.
- =markAsStale()= is not overridden; the base default (emit status + call =loadHistory()=) suffices.
- Diff helpers: =checkString=/=checkBool=/=checkInt= for plain and optional typed fields; inline blocks for tristate (=optional<bool>=) and UUID fields where no base helper exists.

The template no longer contains any =QFutureWatcher=, toolbar, or selection-handling boilerplate.
