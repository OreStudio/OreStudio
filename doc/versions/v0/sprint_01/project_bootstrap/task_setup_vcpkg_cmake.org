* Goal

Wire up CMake with vcpkg so the project can declare external
dependencies and build them as part of the local build, without manual
package installation.

* Acceptance

- A =vcpkg.json= manifest sits at the project root.
- =cmake= configures the project using the vcpkg toolchain.
- A clean build succeeds locally.

* Context

Direct successor to =Create GitHub repository=. Both are prerequisites
for any meaningful project work.
