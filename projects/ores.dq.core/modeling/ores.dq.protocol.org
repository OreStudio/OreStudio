NATS messaging reference for [[id:D3A9F7C2-8B14-4E5D-A6C9-1F7E2B0D8C4A][ores.dq]]. All subjects use the
=dq.v1.>= namespace. See [[id:C9C24C99-F16C-45BE-A262-1C0F4502765E][ores.nats]] for transport details.

Each entity follows the standard pattern: =list=, =save=, =delete=, =history=.

| Subject | Description |
|---------+-------------|

| =dq.v1.catalogs.save= | Create or update a catalog |
| =dq.v1.catalogs.delete= | Delete one or more catalogs |
| =dq.v1.catalogs.history= | Retrieve version history for a catalog |
