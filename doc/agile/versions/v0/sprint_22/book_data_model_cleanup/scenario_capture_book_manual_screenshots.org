* Steps

Each step is its own heading — the title should be short (it's shown
as a single list entry in the QA Validation Runner); put any longer
instructions in the body below the title. The panel writes each
step's PASS/FAIL/PENDING outcome and notes back as a =*** Result=
child heading directly under it.

This is not a pass/fail correctness check — every step's outcome is
"did I save a screenshot with this content to this filename", recorded
as PASS once the file is committed. All four screenshots replace ones
that still show the retired =Trading Book= checkbox and the bogus
5-value =Status= list (Active/Inactive/Closed/Frozen/Pending); the real
values are Status = Active/Closed/Frozen and Regulatory Book Type =
Trading/Banking, each a colour-coded badge combo.

** Capture the Books window list

Open the Books window from the Trading menu. Ensure the visible rows
include at least one book with each Status value (Active, Closed,
Frozen) and each Regulatory Book Type value (Trading, Banking), so
every badge colour is visible in the list. Screenshot the full window,
including toolbar and status bar. Save as
=assets/images/books_main_dialog.png= (overwrite the existing file).

** Capture the Book Details General tab

Open Book Details on any book, with a non-default Status and
Regulatory Book Type selected so both badges are visible in their
closed-box state. Screenshot the General tab in full. Save as
=assets/images/book_details_general.png= (overwrite the existing
file).

** Capture the Status combo open

In Book Details, click the Status combo box open. Screenshot it with
the dropdown showing all three values (Active, Closed, Frozen) as
coloured badges. Save as =assets/images/book_status_combo.png=
(overwrite the existing file).

** Capture the Regulatory Book Type combo open

In Book Details, click the Regulatory Book Type combo box open.
Screenshot it with the dropdown showing both values (Trading, Banking)
as coloured badges. Save as
=assets/images/regulatory_book_type_combo.png= (new file — add the
=[[proj:...]]= image link to replace the placeholder note in
chapter_7_books.org's General tab section once captured).
