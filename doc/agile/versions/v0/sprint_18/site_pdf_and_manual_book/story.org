* Goal

The PDF linked from the Manuals page returns 404 because the site
publish pipeline has no component that copies =.pdf= files into the
output. Additionally, =user_manual.org= uses the =article= LaTeX class
with manual formatting overrides instead of the =book= class. Both
defects are fixed in a single task.

* Acceptance

- =.build-site.el= has a =site:pdf= publish component that copies =.pdf=
  files recursively into the site output directory.
- =https://orestudio.github.io/OreStudio/doc/manual/user_guide/user_manual.pdf=
  is reachable after the next site deploy.
- =user_manual.org= uses =#+latex_class: book= with =oneside= option;
  the =\titleformat{\section}= and =\titleformat{\subsection}= overrides
  are removed.
- The committed =user_manual.pdf= is regenerated with the book class.
