* When to use this skill

When you need to configure, build, test, or deploy ORE Studio using
CMake.

* How to use this skill

1. Identify the operation you need: configure, build, test, deploy
   the site, deploy the skills, or generate diagrams.
2. Open the matching recipe under [[id:5aae6900-8488-4e22-9f31-a1b9d5320efb][CMake setup]] (the knowledge doc) or
   pick from the list below.
3. Run commands from the project root.
4. Default to =linux-clang-debug-ninja= on Linux unless otherwise
   specified.
