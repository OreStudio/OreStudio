* When to use this skill

The search index or org-roam database is stale (compass prints staleness chips) or docs were just added, moved, or renamed.

* How to use this skill

1. /Rebuild both/:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh index --org-roam-db-sync
   #+end_src

   Plain =compass index= refreshes only the FTS cache from an up-to-date org-roam db.
