NATS messaging reference for [[id:D65B6932-7A33-471C-98C6-6AC345D4684C][ores.reporting]]. All subjects use the
=reporting.v1.>= namespace. See [[id:C9C24C99-F16C-45BE-A262-1C0F4502765E][ores.nats]] for transport details.

Each entity follows the standard pattern: =list=, =save=, =delete=, =history=.
