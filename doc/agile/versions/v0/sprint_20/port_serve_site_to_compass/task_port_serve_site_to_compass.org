The command reads =ORES_SITE_PORT= from =_read_env_map()=, kills any process already
on the port via =fuser=, optionally runs =emacs -Q --script ores-build-site.el= with
=--compile=, and serves =build/output/site/= with =http.server.HTTPServer=.  Wired into
the dispatch, =_KNOWN_COMMANDS=, subparsers, and the help epilog.
