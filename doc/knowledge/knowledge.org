Durable, cross-cutting knowledge that outlives any sprint. Each entry
is a single-topic page — a definition, a convention, an explanation of
how some piece of the system works — that other documents can link to
without restating it. If you find yourself explaining the same concept
in two places, that concept belongs here.

are looking for. The [[id:cf2c4f3a-9c5c-4e7a-9e16-2a3a8c1d4c11][external/]] subfolder contains pure reference docs
about systems and concepts ORE Studio does not own.
