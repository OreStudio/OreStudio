* Goal

=compass.sh= sources the =ORES_*= variables from =.env= but not
=SSH_AUTH_SOCK=, so git push/fetch from sandboxed Claude Code sessions
fail until the agent socket is exported by hand — and sourcing the
separate SSH script triggers a permission prompt on every command.
Source =SSH_AUTH_SOCK= from =.env= alongside the other variables so
every compass-driven git operation works out of the box. This matters
more as gh/git operations move into compass (pr/review pillars).
