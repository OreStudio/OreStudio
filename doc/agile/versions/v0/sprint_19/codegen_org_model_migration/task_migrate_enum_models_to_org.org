* Goal

Convert the surviving =workflow_status_enum.json= into a literate
org-mode model under =projects/ores.workflow/modeling/= and delete
the JSON. The other two enums originally listed in the inventory
(=comms/message_type_enum.json=, =comms/compression_type_enum.json=)
are retired separately as part of the comms decommission and no
longer count toward this task.

The migration also has to settle the enum-org *shape* — a different
section layout from the entity meta-model — and extend the codegen
pipeline (org_loader + generator dispatch) to recognise enum org
files.
