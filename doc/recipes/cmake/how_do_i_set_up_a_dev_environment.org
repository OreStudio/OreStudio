* Question

How do I set up a development environment on Ubuntu, Debian, or WSL?

* Answer

1. /Install all required system packages/ with the bootstrap script:

   #+begin_src sh :results verbatim
   ./build/scripts/install_debian_packages.sh --full-install
   #+end_src

   Installs from distro repos: GCC, Clang, CMake, Ninja, PostgreSQL, Qt6,
   Valgrind, and X11/GL headers.

2. /Initialise the database/ before the first build:

   #+begin_src sh :results verbatim
   ./projects/ores.compass/compass.sh db recreate
   #+end_src

3. /Build/ — =compass build= configures automatically on first run
   (see [[id:7C44751E-54C0-4CB7-B489-1D3EC609A4E3][How do I configure the project?]] and
   [[id:F176FCA1-68D5-420D-B076-75A65FEE5EBB][How do I build the system?]]):

   #+begin_src sh :results verbatim
   ./compass.sh build
   #+end_src

* Script

[[proj:build/scripts/install_debian_packages.sh][=build/scripts/install_debian_packages.sh=]] — one-shot package installer.
=compass db recreate= — drops and recreates all ORE Studio
databases with the current schema.

* Tested by

Manual, once per fresh environment. CI uses pre-configured runners and
does not exercise this path.
