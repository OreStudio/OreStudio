* Summary

ORE market data is identified by slash-delimited quote keys of the form
=TYPE/SUBTYPE/DIMENSIONS...=.  The ORE Examples suite
(=external/ore/examples/=) uses two source formats: a legacy whitespace-delimited
=market_YYYYMMDD.txt= and a modern comma-delimited =marketdata.csv=.
Together they cover ten asset classes: interest rates, FX, equity, credit,
commodity, inflation, volatility surfaces (IR, FX, equity, commodity, credit,
bond, inflation), and correlation — 49 distinct quote-key type/subtype pairs
in total.  Every generated synthetic market data environment must populate all
relevant classes in a mutually consistent way; the key cross-asset constraints
are covered in the [[*Cross-asset consistency constraints][Cross-asset consistency constraints]] section.
Return to [[id:7474935C-95D1-3034-B833-83DCD9FB3A09][Knowledge]].

* Quote-key format

#+begin_example
QUOTE_TYPE/SUBTYPE/dim1/dim2/.../dimN  value
#+end_example

Common dimensions (order varies by type):

| Dimension | Meaning | Examples |
|-----------+---------+---------|
| CCY       | ISO currency code | =USD=, =EUR=, =CHF= |
| INDEX     | Rate index or tenor | =3M=, =6M=, =SOFR=, =EURIBOR= |
| TENOR     | Maturity / time to expiry | =1Y=, =10Y=, =2026-06-30= |
| ENTITY    | Equity ticker, credit name, index ID | =SP5=, =BANK=, =2I65BYEG6= |
| STRIKE    | Option strike — absolute, ATM, delta | =ATM=, =ATMF=, =25RR=, =0.05= |
| EXPIRY    | Option expiry (swaption option leg) | =6M=, =10Y= |
| SWAP_LEN  | Swaption underlying swap length | =2Y=, =10Y= |
| INDEX_FAMILY | Inflation index | =EUHICP=, =EUHICPXT=, =AUCPI= |

*** Note on OIS curves
=IR_SWAP= quotes with index tenor =1D= build OIS discount curves (SOFR, ESTR,
SARON, TONAR, SONIA).  No separate quote type; same key schema as par swaps.

** Volatility surfaces (12 types)

All vol surface quotes come in one or more model subtypes.  Subtypes:

| Suffix | Model | Use |
|--------+-------+-----|
| =RATE_LNVOL= | Black log-normal implied vol | Positive-rate regimes |
| =RATE_NVOL= | Normal (Bachelier) implied vol | Low/negative-rate regimes; also natural for spread options |
| =RATE_SLNVOL= | Shifted log-normal vol | Moderate negative rates with shift |
| =SHIFT= | Shift parameter | Used alongside =RATE_LNVOL= to specify the displaced model |
| =PRICE= | Market option price | Used when vol is not directly observable (inflation cap/floor) |

* Cross-asset consistency constraints

Different generation methods per asset class are permitted, but the following
relationships must hold across the generated environment.  Violation of any
constraint will cause ORE to either fail to calibrate or produce obviously
wrong prices.

| Constraint | Relationship | Generation rule |
|------------+-------------+-----------------|
| *Covered interest parity (CIP)* | =FXFWD ≈ FX_SPOT × exp((r_domestic − r_foreign) × T)= | Generate IR curves first; derive FX forward points from spot + CIP.  Never generate =FXFWD= independently. |
| *FX vol and FX spot* | ATM strike of the vol surface = FX forward at each maturity | Vol surface moneyness is relative to the ATM (= FX forward).  If spot changes, re-anchor the surface in strike space. |
| *Equity forward and carry* | =EQUITY_FWD = EQUITY_SPOT × exp((r − q) × T)= | Generate from spot + IR discount curve + dividend yield.  Do not quote forwards independently unless overriding carry. |
| *Equity vol and equity spot* | ATMF strike = forward price, not spot | Re-anchor the vol surface if spot or dividend yield changes. |
| *Swaption vol and IR level* | Normal vols are roughly rate-independent; LN vols scale inversely with rate | In deeply negative-rate environments, always use =SWAPTION/RATE_NVOL= and =CAPFLOOR/RATE_NVOL=.  Apply the same model choice consistently across swaption and cap/floor for the same currency. |
| *Cap/floor shift consistency* | =CAPFLOOR/SHIFT= must exceed =|min forward rate|= | Generate the IR curve first; compute the minimum bootstrapped forward rate; set shift ≥ |min_forward| before generating cap/floor vols. |
| *Inflation curve and seasonality* | Raw index level = smooth ZC inflation level × seasonal factor | =SEASONALITY/RATE/MULT= values must multiply to ≈1.0 over 12 months.  Generate ZC inflation curve first; derive YY rates from ZC; apply seasonal factors consistently. |
| *CDS spread and recovery* | Hazard rate λ ≈ CDS_spread / (1 − Recovery) | Fix recovery first (e.g. flat 40%); generate CDS spreads; never generate both independently. |
| *Commodity forward and spot* | Financial: =COMMODITY_FWD ≈ COMMODITY_SPOT × exp(r × T)=; energy: contango/backwardation is market data | For precious metals enforce carry consistency.  For energy, generate the forward curve directly from a futures strip model; ensure front contract is consistent with quoted spot. |
| *Commodity spread and individual vols* | Spread vol < individual leg vols if correlation > 0 | When generating spread/basis vols (=COMMODITY_OPTION/RATE_NVOL=), ensure they are consistent with the individual commodity vols and the inter-commodity correlation. |
| *Correlation and vol surfaces* | Implied correlation backed out from basket option prices must match =CORRELATION/RATE= | For products with explicit correlation inputs (basket equity options, CMS spread options), generate =CORRELATION/RATE= jointly with the individual vol surfaces, not independently. |
| *Swaption vol and cap/floor vol* | ATM swaption vol and cap/floor vol imply the same forward rate distribution for the relevant tenor | Use a consistent IR model (e.g. LGM or SABR) to generate both; do not generate them as independent random surfaces. |
| *Bond option vol and IR vol* | =BOND_OPTION/RATE_LNVOL= is related to swaption vol via duration | For synthetic generation, derive bond option vols from the swaption vol surface using approximate duration mapping rather than generating independently. |
| *Base correlation and index spread* | Index spread = par-spread of the equity tranche at 100% detachment | Generate the underlying index CDS spread first; derive base correlations to be consistent with the index spread at the equity tranche level. |
