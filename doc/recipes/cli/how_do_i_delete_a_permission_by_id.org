* Question

How do I delete a permission by ID using =ores.cli=?

* Answer

#+begin_src sh :wrap src text
export ORES_CLI_DB_PASSWORD
./ores.cli iam permissions delete ${db_args} ${log_args} \
  --key "currencies:audit"
#+end_src
