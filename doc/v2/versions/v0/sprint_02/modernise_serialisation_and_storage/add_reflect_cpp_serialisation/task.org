* Goal

Integrate reflect-cpp into the build and use it to generate JSON
serialisers for the existing types, replacing the hand-written
serialisation code.

- Existing JSON serialisers replaced by reflect-cpp generated code.
- Round-trip tests pass for all migrated types.

  follow-on task).
