* Goal

Switch CI and dev installs to a prebuilt Qt rather than building Qt from vcpkg from source.

* Acceptance

- Prebuilt Qt detected if present; vcpkg Qt build skipped.
- Fallback to vcpkg build documented for environments without the prebuilt.
- CI runtime drops measurably.

* Plan

Captured during execution; cleared into the parent story on close.

* Notes

Qt-from-source dominated CI runtime; this was the single biggest win available.

* Result

CI and local builds substantially faster.
