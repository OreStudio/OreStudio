An ISO 3166-1 country identified by its two-letter alpha-2 code (e.g. =GB=). Each
country carries alpha-2, alpha-3, and numeric codes, a name, an official name, an
optional coding-scheme provenance tag, and an optional flag image reference.
Countries are tenant-scoped reference data (scope_both: validates against the
tenant's own set and the system tenant's canonical set). Used as FK reference from
party–country junctions and anywhere a country is recorded.

The following deviations from what the template generates are intentional and

| Unique index name | =countries_alpha2_code_uniq_idx= | =countries_code_uniq_idx= | Renamed to match template; no application code references index names |
| security definer placement | footer of function body: =$$ language plpgsql security definer set search_path...= | header before =as $$= | Adopted template form — both are valid PostgreSQL |
