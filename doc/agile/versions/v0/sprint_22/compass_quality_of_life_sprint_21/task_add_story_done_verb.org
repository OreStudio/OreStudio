* Result

Delivered on =feature/add-story-done-verb=.

Added =_cmd_story_done(story_ident)= in =compass.py= and wired it as
=compass story done <slug-or-uuid>=.

Behaviour mirrors =compass task done=:

- Resolves the story by UUID (exact or prefix) or folder slug.
- Guards: if any child task is not DONE or ABANDONED, prints the
  offenders and exits 1.
- Flips story doc State → DONE; stamps Now and Next to "Nothing."
- Updates the sprint's =* Stories= table row via the existing
  =_update_task_row_in_story= helper (same UUID-based row lookup,
  sets end date).
- Stamps the journal and prints a reminder to write =* Result= prose.
