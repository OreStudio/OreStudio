* Goal

Running ores.shell against the local environment requires hand-rolled
NATS flags and credentials. =compass shell= resolves all of it from
=.env=: binary path from the preset, NATS url/prefix/TLS, and login
credentials; =-f FILE= feeds a scripted session; logging flags are
forwarded. The Emacs dashboard's shell launcher goes through it, so
every entry point shares one definition of "connect to this checkout".

Merged in [[https://github.com/OreStudio/OreStudio/pull/1097][PR 1097]]. compass shell launches ores.shell with the binary
resolved from the checkout's preset and the NATS connection and login
credentials from .env passed as explicit flags; -f runs scripted
sessions (exit auto-appended), -u/-p override credentials, logging
flags forward, --dry-run masks the password, -- passes through. The
Emacs dashboard's shell launcher goes through it. Review round 1
