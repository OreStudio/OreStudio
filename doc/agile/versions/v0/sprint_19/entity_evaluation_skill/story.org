* Goal

A Claude Code skill and companion runbook exist for evaluating the health of any entity: the
skill looks up coverage from [[id:A3F8C2D1-7E4B-4F9A-B2C5-8D1E6A3F0B7C][Entity Coverage Matrix]], walks through standard functional
verification steps (load, edit, save, events, history), and produces a concise gap report.
This replaces ad-hoc manual investigation with a repeatable, automated procedure.
