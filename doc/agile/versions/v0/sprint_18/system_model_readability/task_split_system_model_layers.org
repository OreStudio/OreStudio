* Goal

=projects/modeling/system_model.org= has grown large: it contains the full
sub-component tables for all four architecture layers in one file. This task
splits each layer into its own dedicated org-mode document, linked back from
the main page via org-roam ID links. It also adds a PlantUML diagram
illustrating the four-layer architecture (Foundation → Infrastructure →
Domain → Application) and wires in org-roam links wherever [[id:C9C24C99-F16C-45BE-A262-1C0F4502765E][ores.nats]] is
mentioned as a plain-text "NATS" reference. Finally it merges the
now-superseded =site_pdf_and_manual_book= story into =user_manual_pdf_build=
by moving its task file there, deleting the old story directory, and updating
all org-roam cross-references.
