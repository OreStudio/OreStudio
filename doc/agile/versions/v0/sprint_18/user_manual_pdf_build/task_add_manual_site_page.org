* Goal

The existing =user_manual.org= is a single large org-mode file suitable
for LaTeX/PDF export but produces one very long HTML page on the site.
This task adds a parallel =user_manual_site.org= — a lightweight HTML
entry point that gives readers a short overview and a chapter-by-chapter
table with org-roam links to each chapter file. It also restructures
=manuals.org= from prose sections into a clean table, linking to the new
site page (not the monolithic file), and updates the chapter 1
description to be more informative.
