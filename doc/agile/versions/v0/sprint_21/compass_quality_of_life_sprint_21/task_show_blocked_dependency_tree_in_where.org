* Result

Delivered on =feature/show-blocked-dependency-tree-in-where=.

Added a "🔴  Blocked (N)" section to =cmd_where= in =compass.py=, printed
after "In flight" and before the PRs section.

For each BLOCKED task in the current sprint:
- Prints the task title with a 🔴 icon and optional "since <date>" note
  (from =#+blocked_since:=).
- Walks the =#+blocked_on:= UUID chain up to 8 hops, rendering an
  indented =↳= tree.  Each hop shows the blocker's current state icon
  (🔴 BLOCKED, ✅ DONE, ▶️ otherwise), state label, title, and
  =compass show= command.
- Guards against cycles (seen-UUID set) and missing UUIDs
  (prints ❓ with the short UUID).
- Section is omitted entirely when no BLOCKED tasks exist.
