List countries via =ores.cli=. See the [[id:A1B2C3D4-E5F6-7890-ABCD-EF1234567890][CLI inventory]] for sibling recipes.

* Question

How do I list countries?

* Answer

** List all as JSON

#+begin_src sh
export ORES_CLI_DB_PASSWORD
./ores.cli refdata countries list ${db_args} ${log_args} --format json | jq .[:3]
#+end_src

** List all as table

#+begin_src sh :wrap src org
export ORES_CLI_DB_PASSWORD
./ores.cli refdata countries list ${db_args} ${log_args} --format table | head -n 15
#+end_src
