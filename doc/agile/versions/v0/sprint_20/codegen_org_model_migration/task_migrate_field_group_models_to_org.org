* Goal

Convert the 5 =trade_*_field_group.json= files under
=projects/ores.codegen/models/trading/= into literate
=ores.trading.<name>_field_group.org= files under
[[proj:projects/ores.trading/modeling][=projects/ores.trading/modeling/=]]. Field-group models are small
plain-struct definitions used to compose =trade= via
=rfl::Flatten<T>= (the workaround for MSVC C1202 too-many-fields
errors). Extend the org pipeline to recognise the new kind:
=#+type: ores.codegen.field_group=, a filename suffix
=_field_group.org= to discriminate from entity org files, and a
dedicated loader.
