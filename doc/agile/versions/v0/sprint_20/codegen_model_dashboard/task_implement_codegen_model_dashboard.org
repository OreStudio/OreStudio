* Acceptance

- Dashboard loads at =/OreStudio/codegen/= (GitHub Pages) and locally.
- 127 codegen nodes displayed as colour-coded circles; links as grey lines.
- Legend maps colour → type (generator-config types grouped as one entry).
- Clicking a node opens sidebar: type badge, title, description, link chips.
- Template org files surface as chips in the sidebar of nodes that link to
  them; clicking a chip opens the template's sidebar with a collapsible
  Mustache source pane.
- Only codegen-relevant nodes appear; all other documents are excluded.
- Deploy wired via =ores-deploy-web-apps= in =ores-build-site.el=.
