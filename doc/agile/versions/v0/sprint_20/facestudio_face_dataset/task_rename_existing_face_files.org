* Goal

The 60 images in =external/facestudio/faces/= were downloaded before the
script was fixed. They use the old naming scheme (=female_age25_african.jpeg=)
and include out-of-range ages (00–15, 75–80). This task brings the existing
files into conformance with the fixed spec so that new downloads continue
from where we left off without duplication:

1. Rename each valid file from =<gender>_age<NN>_<ethnicity>.jpeg= to
   =<gender>_age<NN>_<ethnicity>_01.jpeg=.
2. Delete any file whose age component is below 20 or above 70.
