How do I delete a feature flag by key using =ores.cli=?

* Answer

#+begin_src sh  :wrap src text
export ORES_CLI_DB_PASSWORD
./ores.cli variability feature-flags delete ${db_args} ${log_args} \
  --key "Test feature3"
