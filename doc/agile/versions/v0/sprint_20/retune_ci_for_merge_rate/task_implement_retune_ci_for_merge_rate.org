* Goal

Create =pr.yml=: a classifier job that diffs the PR's changed paths
into docs/code/ci classes, conditional jobs per class (docs → site
build; code → canary + roundtrip; ci → everything), and a =pr-gate=
summary job that always reports and becomes the single required
check. Fold the roundtrip check (misnamed =ore_coverage.yml=) into
the code class; retire the standalone PR triggers the classes
replace.
